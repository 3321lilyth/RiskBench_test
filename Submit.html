  • Register a user & team name
  • Prepare your test case answer and name it with result_interactive.txt, the format of result_interactive.txt should be
    [scenario name]#[variant name]#[variant attribute value] [if collision] [smallest distance], where

    • scenario name , variant name and variant attribute value are text that describe the scenario truth,
    • if collision is a bool indicate if there is a collision in this scenario,
    • smallest distance is the smallest distance between the ego-car and the interactor during the whole journey.

    Here is an example of result_interactive.txt:
    10_t3-6_1_b_l_l_1_0#WetNoon#high False 8.193608817278493
    10_t3-6_1_b_l_l_1_0#ClearNight#high False 8.170071255258556
    10_t3-6_1_b_l_l_1_0#HardRainSunset#low False 8.187130409417815
    10_t3-6_1_b_l_l_1_0#MidRainSunset#mid False 8.245623946608797
    10_t3-6_1_b_l_l_1_0#WetNight#high False 8.24798449033569
    ...

  • Submit result_interactive.txt using the button below, you can give every submission a description to distinguish them.
